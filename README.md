# bandishen.com
a repo for placing bandishen and their respective ragas

```
Folder structure
|
|
|- [Thaat, eg Bilawal, Todi, etc]
    |- [Raga name, eg Yaman, Kafi, etc]
    |- [Bageshrii]
    |- [Bheem_Palasi]
    |- [Bandishen of that respective Raga]
|- [Thaat, eg Bilawal, Todi, etc]
    |- [Raga name, eg Yaman, Kafi, etc]
    |- [Durga]
    |- [Bandishen of that respective Raga]
|
|
```
